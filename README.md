Simple Calculator 

A Simple Calculator built with HTML, CSS, and JavaScript. This project demonstrates DOM manipulation, event handling, and using localStorage for saving calculation history. 
The calculator supports both mouse clicks and keyboard input,making it interactive and user-friendly.

Features

- Perform basic arithmetic operations: addition, subtraction, multiplication, division.
- Clear display with "C" button or Escape key.
- Backspace support using the Backspace key.
- Calculate results using = button or Enter key.
- Calculation history: stores last 10 calculations using `localStorage`.
- Clear history button to remove all past calculations.
- Fully keyboard-compatible.

Technologies Used

- HTML
- CSS
- JavaScript (DOM manipulation, events)
- Browser `localStorage` for persistent history

Usage

1. Clone this repository or download the files.  
2. Open `index.html` in your web browser.  
3. Use the calculator with:
   - Mouse clicks on buttons, **or**
   - Keyboard input:
     - Numbers `0-9`  
     - Operators `+ - * /`  
     - `Enter` for equals  
     - `Escape` to clear  
     - `Backspace` to delete last character  
4. View calculation history in the History section.  
5. Clear history by clicking the Clear History button.
